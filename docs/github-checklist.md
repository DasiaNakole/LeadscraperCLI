# GitHub Checklist

Before publishing this repo:

- keep `.venv/` out of the repo
- keep generated one-off output like `demo_leads.csv` out of the repo
- include `README.md`
- include `sample_leads.csv`
- include tests
- include fixture HTML used for demo mode

## Recommended Repo Name

- `lead-scraper`
- `business-lead-scraper`
- `prospect-list-builder`

## Recommended First Commit Message

`Initial commit: portfolio-ready lead scraper CLI with CSV export`

## Recommended GitHub Extras

- add a short project description
- pin the repo on your profile
- add a Loom demo link to the README
- add 2 to 3 screenshots of terminal usage and CSV output
